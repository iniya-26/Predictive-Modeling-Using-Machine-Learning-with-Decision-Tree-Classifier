# Predictive Modeling Using Machine Learning with Decision Tree Classifier

## 📌 Project Overview

This project demonstrates a complete machine learning workflow using the Titanic dataset. The objective is to build a predictive model that determines whether a passenger survived the Titanic disaster based on various passenger attributes. The project includes data preprocessing, feature selection, model training, evaluation, and prediction using the Decision Tree Classifier.

---

## 🎯 Objective

To develop a machine learning model capable of predicting passenger survival by analyzing important features from the Titanic dataset.

---

## 📂 Dataset

**Dataset:** Titanic Dataset

**Features Used**
- Passenger Class (Pclass)
- Gender (Sex)
- Age
- Fare
- Number of Siblings/Spouses (SibSp)
- Number of Parents/Children (Parch)
- Embarked

**Target Variable**
- Survived

---

## 🛠️ Technologies Used

- Python
- Google Colab
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

---

## 🧹 Data Preprocessing

The following preprocessing steps were performed:

- Removed duplicate records
- Filled missing values using median and mode
- Converted categorical variables into numerical values
- Selected important features for training
- Split the dataset into training and testing sets

---

## 🤖 Machine Learning Algorithm

**Decision Tree Classifier**

The model was trained using an 80:20 train-test split to predict passenger survival.

---

## 📊 Model Evaluation

### Accuracy
**79.89%**

### Evaluation Metrics

- Accuracy Score
- Classification Report
- Confusion Matrix
